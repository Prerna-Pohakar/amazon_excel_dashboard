# amazon_excel_dashboard
Amazon Sales & Customer Insights Dashboard (Excel)

This project is my first data analytics dashboard, created using Microsoft Excel.
It provides a clear overview of Amazon’s sales performance, top-selling categories, discount impact, and customer insights using charts and pivot tables.

Key Features

1.Total Revenue Analysis (₹45,78,580)

2.Top-Selling Product Categories

3.Revenue Contribution by Category

4.Discount % vs Actual Price Comparison

5.Average Product Rating & Customer Insights

Skills Used

1.Excel (Pivot Tables, Data Cleaning, Charts, Formatting)

2.Data Visualization

3.Basic Sales & Customer Analytics
